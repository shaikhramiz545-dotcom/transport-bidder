# Updates Log

This folder contains a short log entry for every fix or new feature across:

- user_app
- driver_app
- admin_panel
- agency_portal
- corporate_portal
- backend

## Rules

- Create one file per change set.
- File name format:
  - `YYYY-MM-DD-short-title.md`
- Keep entries short and practical.
